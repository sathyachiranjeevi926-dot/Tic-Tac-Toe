# Tic-Tac-Toe AI (Python)

A console-based Tic-Tac-Toe game developed in Python where a player competes against a computer-controlled opponent.
The game features a rule-based AI that makes strategic decisions by prioritizing winning moves, blocking the player's winning opportunities, and selecting optimal positions whenever possible.

## Features

* Interactive command-line interface
* Smart AI with attack and defense strategies
* Automatic win, loss, and draw detection
* Input validation for player moves
* Clean and modular code using functions
* Simple and lightweight implementation using only Python's standard library

## Technologies Used

* Python 3
* Random module

## AI Strategy

The computer opponent follows a rule-based decision-making approach:

* Takes the center position whenever available.
* Completes a winning move if possible.
* Blocks the player's immediate winning moves.
* Selects the next available strategic position when no attack or defense is required.
